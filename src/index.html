Installation Guide

Follow these steps to install APEX_MVP in Microsoft Word:

Word Desktop
Word Online
Troubleshooting
1
Download the Manifest

Right-click on manifest.xml and select "Save link as..." to download the file to your computer.

2
Open Microsoft Word

Launch Microsoft Word on your desktop computer.

3
Access Add-ins

Go to Insert tab → Add-ins group → My Add-ins → Upload My Add-in

4
Upload the Manifest

Click Browse and select the downloaded manifest.xml file, then click Upload.

5
Start Using

Look for "APEX_MVP" in the Home tab ribbon and click "Show APEX_MVP" to open the taskpane.

Note: If you see a security warning, click "Trust this add-in" to proceed.
1
Open Word Online

2
Access Add-ins

3
Upload Manifest

Click Browse and select the manifest.xml file, then click Upload.

Important: Word Online may have limited add-in support. For the best experience, use Word Desktop.

Common Issues

Add-in not appearing in ribbon

Solution: Try restarting Word after uploading the manifest. The add-in should appear in the Home tab under "APEX_MVP".

Security warning

Solution: Click "Trust this add-in" when prompted. This is normal for custom add-ins.

Upload fails

Solution: Ensure you're using a supported version of Word (2016 or later) and that the manifest.xml file is not corrupted.

Taskpane doesn't load

Solution: Check your internet connection. The add-in loads content from GitHub Pages.

Need Help? Create an issue on our GitHub repository for support.

How to Use

  1. Install the Add-in: Follow the installation steps above
  2. Open Word: Launch Microsoft Word (desktop or online)
  3. Load the Add-in: Upload the manifest.xml file
  4. Access Taskpane: Click the 'Show APEX_MVP' button in the ribbon
  5. Start Using: Select text and use the AI-powered features
Note: This is a development version. For production use, ensure proper testing and validation.
