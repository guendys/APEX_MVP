APEX_MVP Commands

Available Commands and Features

Status: ✅ Commands are ready for use in Microsoft Word
Primary Command: Show Taskpane
This is the main command that opens the APEX_MVP taskpane in Microsoft Word. The taskpane contains all the AI-powered features and tools.
Ribbon Location: Home Tab → APEX_MVP Group → "Show APEX_MVP" Button
Taskpane Features
Once the taskpane is open, you'll have access to these powerful features:
  • Smart Edit Suggestions: Get AI-powered suggestions for improving your text
  • Document Analysis: Evaluate sentences and get feedback
  • Live Sentence Mode: Real-time suggestions as you type
  • Content Generation: Generate new content based on your document
  • RAG Analysis: Retrieve and analyze information from your document
  • Project Chat: Ask questions about your project
  • Feedback System: Provide feedback on suggestions
How to Use
  1. Install the Add-in: Load the manifest.xml file in Word
  2. Open Word: Launch Microsoft Word (desktop or online)
  3. Find the Command: Look for "APEX_MVP" in the Home tab ribbon
  4. Click the Button: Click "Show APEX_MVP" to open the taskpane
  5. Start Using: Select text and use the AI features
Technical Details
